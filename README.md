# LeetCode Problem Solutions 🧩

A personal collection of LeetCode problem solutions written in **C++**, built as part of my journey to sharpen my data structures & algorithms skills and prepare for software engineering interviews in the global job market.

## 📌 About

This repo is where I track my problem-solving practice — from fundamentals to more advanced patterns. Each solution includes clean, commented C++ code, and (where useful) a short note on the approach and time/space complexity.

## 📁 Repository Structure

Problems are organized by difficulty:

```
LeetCode-Problem-Solutions/
├── Easy/
│   ├── 0001-two-sum.cpp
│   ├── 0009-palindrome-number.cpp
│   └── ...
├── Medium/
│   ├── 0003-longest-substring-without-repeating-characters.cpp
│   └── ...
├── Hard/
│   ├── 0042-trapping-rain-water.cpp
│   └── ...
└── README.md
```

Each file is named using the pattern `<problem-number>-<problem-title>.cpp` for easy reference.

## 🚀 How to Use

1. Clone the repo:
   ```bash
   git clone https://github.com/<your-username>/LeetCode-Problem-Solutions.git
   ```
2. Navigate to a difficulty folder and open any `.cpp` file.
3. Compile and run locally to test:
   ```bash
   g++ filename.cpp -o solution && ./solution
   ```
